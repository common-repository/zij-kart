Zij kart. Let you create your online shop easily and elegantly.

== Description ==
Zij kart provide the easy and best solution for the online showroom. You can categorized your product. You can easily manage your product category, brand and products with different color options while uploading images for your product.

= Major Features =

* Categories for products
* Brands for products
* Paypal payment method
* Purchase history with detail
* Product images with color options
* Shortcodes for the products [zijkart_products]
* Shortcodes for the brands [zijkart_brands]
* Shortcodes for the categories [zijkart_categories]
* Shortcodes for the cart [zijkart_cart]

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add widget from the appearance->widget to your favorite location where you want to show indeed jobs into your theme

After plugin activation goto pages and add page for the products add the shortcode for the products and publish it. Then goto zij kart => options and select your page title in Default page combo box and save options that's it now you have working zij kart.

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.2.4 or greater
